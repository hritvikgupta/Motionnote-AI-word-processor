# Getting Started with MotionNote

Welcome to MotionNote! This guide will help you get started with the AI-powered word processor.

## üöÄ Quick Start

### 1. Try the Demo

No sign-up required! Experience MotionNote right now:

1. Visit [motionnote.com](https://motionnote.com) (or your production URL)
2. Scroll to the demo section
3. Try the AI Researcher, Annotate, and Knowledge Base features
4. Explore the editor capabilities

### 2. Create an Account

To save your work and access all features:

1. Click "Get Started" or "Sign in" in the navigation bar
2. Choose your sign-up method:
   - **Email/Password**: Enter your email and create a password
   - **Google OAuth**: Sign in with your Google account
3. Verify your email (if using email/password)
4. Start creating!

## üìù Your First Document

### Creating a New Document

1. After signing in, you'll see your dashboard
2. Click "New Document" or "Create"
3. Start typing or use AI features to generate content

### Using AI Researcher

1. Open or create a document
2. Click the "AI Researcher" button (green button with sparkle icon)
3. Ask questions about your document:
   - "Summarize this document"
   - "What are the main points?"
   - "Improve this section"
4. Get AI-powered insights and suggestions

### Using AI Annotations

1. Write or paste your content
2. Click the "Annotate" button
3. Wait for AI analysis
4. Review suggestions and apply improvements

### Using Knowledge Base

1. Click the "Knowledge Base" button
2. Upload files you want to reference
3. Use these files with AI Researcher for context-aware responses

## üéØ Common Tasks

### Formatting Text

- **Bold**: Select text and click Bold (B) or use `Ctrl+B` / `Cmd+B`
- **Italic**: Select text and click Italic (I) or use `Ctrl+I` / `Cmd+I`
- **Headings**: Use the heading dropdown or Markdown syntax (`# Heading`)

### Adding Images

1. Click the image icon in the toolbar
2. Upload from your computer or paste a URL
3. Resize and position as needed

### Creating Tables

1. Click the table icon in the toolbar
2. Select the number of rows and columns
3. Start editing cells

### Exporting Documents

1. Click "Export" in the menu
2. Choose your format (PDF, Word, HTML, etc.)
3. Download your document

## üí° Tips & Best Practices

### Maximizing AI Features

- **Be specific**: The more context you provide, the better AI responses
- **Use Knowledge Base**: Upload reference materials for better AI understanding
- **Review annotations**: AI suggestions are helpful, but always review before applying

### Organizing Your Work

- Use clear document titles
- Organize documents into folders (if available)
- Use tags or categories for easy searching

### Performance Tips

- Large documents work great, but very large files may take longer to process
- Use pagination mode for better performance with long documents
- Save frequently (auto-save is enabled by default)
